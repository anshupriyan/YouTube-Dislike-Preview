# Security Policy

## Reporting Security Concerns
If you discover a security vulnerability or privacy issue, please report it privately rather than opening a public issue.

* **GitHub Private Vulnerability Reporting:** Use the **"Report a vulnerability"** feature under the [Security tab](https://github.com/anshupriyan/YouTube-Dislike-Preview/security/advisories/new) of this repository.

## Scope & Data Privacy
* **Scope:** YouTube Dislike Preview is a lightweight browser extension that reads YouTube page elements to display dislike statistics and fetches rating metrics from the public Return YouTube Dislike API (`returnyoutubedislikeapi.com`).
* **Data Privacy:** The extension does **not** collect, track, or transmit any personal browsing data, account information, user identifiers, or IP addresses.

## Response Expectations
* **Response Time:** This is a solo-maintained, best-effort open-source project. Security reports will be reviewed as soon as possible.
* **Bug Bounty:** There is currently no formal bug bounty program or monetary compensation for vulnerability disclosures.
