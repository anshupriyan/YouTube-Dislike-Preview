# Privacy Policy for YT Dislike Preview

YT Dislike Preview is committed to protecting your privacy. This Privacy Policy describes how we handle information when you use our Chrome Extension.

## 1. Information Collection and Use
* **No Personal Data Collection:** The extension does not collect, store, or transmit any personally identifiable information (PII) such as names, email addresses, IP addresses, or browsing history.
* **No Tracking:** We do not track user activity, search queries, or mouse clicks. No telemetry or analytic tools are included in the extension.

## 2. Third-Party API Interaction
* **Data Retrieval:** To fetch dislike and like counts, the extension communicates with the public Return YouTube Dislike API (`https://returnyoutubedislikeapi.com`). 
* **Transmitted Data:** The extension only sends the 11-character YouTube Video ID (e.g. `dQw4w9WgXcQ`) to the API. No user identifiers, cookies, or account details are sent with these requests.

## 3. Changes to This Policy
We may update our Privacy Policy from time to time. Any changes will be posted directly on this page.

## 4. Contact
If you have any questions or feedback about this privacy policy, feel free to open an issue on our GitHub repository: [GitHub Repository](https://github.com/anshupriyan/YT-Dislike-Preview).
